PDFRpt Reporting API

PDFRpt is designed for quick and easy generation of PDF reports in Java. It relies heavily on iText, which provides a complete Java API for any and all types of PDF manipulation in Java.

System Requirements

PDFRpt requires Java, version 1.5 or greater.

Download

Current version is 1.0.1.

Download the compiled distribution.

Download the source code from this site, or from GitHub

Using it

Three jar files are included with the distribution, and all should go in your Java classpath. The iText-5.0.3.jar can probably be replaced with the most current version of iText, assuming such is backwards compatible. Beyond that, refer to the JavaDoc API link below.
