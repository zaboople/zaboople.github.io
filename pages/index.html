Welcome to my web site. You are likely here because you found it on my resume. This site hosts several semi-experimental projects that I have worked on over the years, and includes all of the source code, documentation and binary distributions.

Currently hosted are three projects. The first was the original project hosted on this site, which thus bears its name. Altogether they are:

  • Tmplz, an API that can be used to generate web pages or any other kind of text file from templates;
  • PDFRpt, an API for PDF report generation;
  • Klonk, a text editor featuring an unusual undo/redo algorithm that preserves all edit states.

Although source code is available directly on this site, you can also peruse it on GitHub.
